⚡ ゼロトークン印刷

curlでアップロード、MCPで印刷指示。base64不要、100ページPDFでもトークン消費ゼロ。

# ステップ1: bash_toolでアップロード curl -sF "file=@/mnt/user-data/uploads/report.pptx" https://printer-mcp.appserver.tokyo/upload → {"file_id":"abc123"} # ステップ2: MCPツールで印刷 print_uploaded(file_id="abc123", cups_options={"Stpl":"DualLeft","Scnt":"All"})

🚀 セットアップ（5ステップ）

Proxmox LXC + CUPS + Kyoceraドライバ + Cloudflare Tunnel の構成。

Proxmox LXCコンテナ作成
Ubuntu 24.04テンプレートからクローン。2コア/4GB/20GB。
CUPS + Kyoceraドライバ + プリンタ登録
CUPS, Kyocera UPD v10.0, lpadminでsocket://登録。フィニッシャー設定（DF-7150, Punch, Fold, Inserter）。PPDカスタマイズ（DualLeft等のステープル追加）。
MCPサーバーデプロイ
Node.js 22 → git clone → npm install → systemd。環境変数にMCP_API_KEY, GRAPH_*。
Cloudflare Tunnel
cloudflared → hostname → http://localhost:3000。
Claude.aiに接続
Settings → MCP → Add。URL: https://printer-mcp.your-domain/mcp?key=KEY
