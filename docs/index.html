⚡ ゼロトークン印刷

ファイルをcurlで直接アップロード。base64エンコード不要。100ページのPDFでもトークン消費ゼロ。

# ステップ1: bash_toolでアップロード（トークン消費ゼロ） curl -sF "file=@/mnt/user-data/uploads/report.pptx" https://printer-mcp.appserver.tokyo/upload → {"file_id":"abc123", "filename":"report.pptx", "size":2450000} # ステップ2: MCPツールで印刷（cups_optionsの数十トークンのみ） print_uploaded(file_id="abc123", cups_options={"Duplex":"DuplexNoTumble"})
📄
ファイル形式を意識不要
PDF送ろうがPPTX送ろうがDOCX送ろうが同じ操作。LXC内で拡張子を見て自動判定・変換・印刷。
☁️
Office自動変換
DOCX/XLSX/PPTXはGraph API経由でPDFに自動変換。AI側は何もしなくていい。
🗑️
30分自動削除
アップロードファイルは/tmp/に一時保存。印刷後または30分経過で自動削除。

💬 使い方

自然言語で指示するだけ。AIがトレイ確認→アップロード→制約検証→印刷→完了確認まで一貫実行。

ユーザー: この報告書をA4両面で左上ステープル、2穴パンチで10部印刷して AI: get_printer_status → idle, カセット1: A4普通紙 ✓ bash_tool: curl -sF "file=@report.docx" .../upload → file_id: abc123 validate_print_options → ✅ 2,134ルール検査、違反0 print_uploaded(abc123, {Stpl:Front, Scnt:All, Pnch:2Hole, Duplex:NoTumble}) → report.docx → Graph API変換 → PDF → CUPS → Job #42 get_job_status(42) → ✅ completed
ユーザー: この横長スライドをA4縦に2スライドずつ配置して中綴じ製本にして AI: bash_tool: curl -sF "file=@slides.pptx" .../upload → file_id: def456 validate_print_options({number-up:2, KCBooklet:Left, Fold:True, PageSize:A4}) → ✅ 有効 (number-up: CUPS built-in, KCBooklet: PPD validated) print_uploaded(def456, {number-up:2, KCBooklet:Left, Fold:True, PageSize:A4}) → PPTX → PDF → CUPS 2up面付け → ドライバ冊子面付け → A3紙 → 中綴じ → A4冊子

🛡️ PPD制約バリデーション

3,192件のUIConstraintsルール + CUPSビルトインオプション対応。印刷前に自動検証。

❌ 中綴じ + コーナーステープル
ステープル(Front) と 中綴じ製本(Left) は同時に指定できません
❌ A5 + 二つ折り
折りモード(Bifold) と 用紙サイズ(A5) は同時に指定できません
✅ 2up面付け + 中綴じ製本
全て有効。number-up: CUPS built-in, KCBooklet/Fold/PageSize: PPD validated
✅ A4両面 + ステープル + パンチ
全て有効。2,134ルール検査済み、違反0

🚀 セットアップ（5ステップ）

Vercelワンクリックではありません。Proxmox LXC + CUPS + Kyoceraドライバ + Cloudflare Tunnel の構成。

Proxmox LXCコンテナ作成
Ubuntu 24.04テンプレートからクローン。2コア/4GB/20GB。onboot=1。
CUPS + Kyoceraドライバ + プリンタ登録
CUPS, Kyocera UPD v10.0, lpadminでsocket://プリンタIP:9100登録。フィニッシャー設定（DF-7150, Punch, Fold, Inserter）。
MCPサーバーデプロイ
Node.js 22 → git clone → npm install → npm run build → systemdサービス化。環境変数にMCP_API_KEY, GRAPH_*を設定。
Cloudflare Tunnel
cloudflaredインストール → トンネル作成 → hostname → http://localhost:3000。
Claude.aiに接続
Settings → MCP → Add。URL: https://printer-mcp.your-domain/mcp?key=KEY
