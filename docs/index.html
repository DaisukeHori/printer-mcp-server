📎 フィニッシャー機能詳細

cups_optionsで全機能を制御。descriptionに全オプション・制約・組み合わせ例を焼き込み済み。

中綴じ製本

// A4中綴じ製本（左綴じ・横書き） → A3紙に2面付け → 折り → A4冊子 {"KCBooklet":"Left", "Fold":"True", "PageSize":"A4"} // B5中綴じ製本（右綴じ・縦書き） → B4紙使用 {"KCBooklet":"Right", "Fold":"True", "PageSize":"B5"}

⚠ PageSizeは仕上がりサイズ。A4→A3紙、B5→B4紙を自動使用。最大20枚(80p)、普通紙のみ。

三つ折り・二つ折り

// A4三つ折り（宛名が外側に見える、DM封筒用） {"FldA":"Trifold", "FldB":"FPInside", "FldC":"RIGHTL", "OutputBin":"FLDTRAY", "PageSize":"A4"}

ステープル＋パンチ

// A4両面 + 左上ステープル + 2穴パンチ {"Stpl":"Front", "Scnt":"All", "Pnch":"2Hole", "Duplex":"DuplexNoTumble", "PageSize":"A4"}

🛡️ PPD制約バリデーション

Kyocera PPDファイルから抽出した3,192件のハードウェア制約ルールで、印刷前にオプション互換性を自動検証。

❌ 中綴じ + コーナーステープル
ステープル(Front) と 中綴じ製本(Left) は同時に指定できません
❌ A5 + 二つ折り
折りモード(Bifold) と 用紙サイズ(A5) は同時に指定できません
❌ 厚紙 + パンチ
用紙種類(Cardstock) と パンチ(2Hole) は同時に指定できません
✅ A4両面 + ステープル + パンチ
全て有効。印刷できます。（2,134ルール検査済み、違反0）

10の重要ルール

1. 中綴じとステープルは同時不可（中綴じは自動ステープル）
2. 折り使用時は OutputBin=FLDTRAY 必須
3. A5以下はステープル/折り不可
4. 封筒/ラベル/OHPは後処理全不可
5. 厚紙はステープル/パンチ/折り全不可
6. 中綴じPageSizeは仕上がりサイズ（A4→A3紙自動）
7. 三つ折りはA4/Letterのみ
8. 中綴じ最大20枚(80p)、折り最大3枚、普通紙のみ
9. 印刷前に validate_print_options で必ず検証
10. get_printer_status でトレイ紙設定を確認してから印刷

🚀 セットアップ

Proxmox LXC + CUPS + Kyoceraドライバ + Cloudflare Tunnel

1️⃣
LXCセットアップ
Proxmox上にUbuntu LXCを作成。CUPS、Kyocera UPDドライバ、Node.js 22をインストール。プリンタ登録＋フィニッシャー設定。
2️⃣
MCPサーバー起動
リポジトリをクローン → npm install → npm run build → systemdサービス化。Graph API環境変数を設定。
3️⃣
Cloudflare Tunnel
cloudflaredをインストール。printer-mcp.your-domain → localhost:3000 のトンネルを作成。
4️⃣
Claude.aiに接続
Settings → MCP → Add。URLにエンドポイント＋APIキーを設定。すぐに使い始められます。
